# Expense-Tracker
A modern Expense Tracker built with HTML, CSS, and JavaScript featuring expense management, budget tracking, category filtering, search functionality, monthly summaries, dashboard cards, and local storage support.
## Project Overview

Expense Tracker is a responsive web application that helps users manage their daily expenses efficiently. Users can add, edit, delete, search, and filter expenses by category. The application includes budget tracking with alerts, monthly spending summaries, dashboard cards, and persistent data storage using the browser's Local Storage API.

### Features

- Add, edit, and delete expenses
- Search expenses by name
- Filter expenses by category
- Set and monitor monthly budgets
- Budget exceeded and remaining budget alerts
- Monthly spending summary
- Dashboard cards for quick insights
- Data persistence using Local Storage
- Responsive design for desktop and mobile devices

### Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
