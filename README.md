# Expense Tracker

A modern and responsive Expense Tracker built using HTML, CSS, and JavaScript.

## Features

- Add Expenses
- Edit Expenses
- Delete Expenses
- Search Expenses
- Filter by Category
- Budget Tracking
- Budget Alerts
- Monthly Summary
- Dashboard Cards
- Local Storage Support
- Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
